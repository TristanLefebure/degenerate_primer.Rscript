# degenerate_primer.Rscript

A short R script to produce all the combination of IUPAC sequences, typically degenerated primers.

GPL Licence

Exemple using Vasco Elbrecht and Florian Leese COI BF2 primer :

``
./degenerate_primer.R BF2 GCHCCHGAYATRGCHTTYCC
``

Will produce a fasta file (BF2_degenerated.fas) with 216 primers:
